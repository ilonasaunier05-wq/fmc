API Vault

Optional free keys for richer Smart Scroll

Keys are stored only in this browser’s local storage on your phone. Do not paste paid/secret keys into a shared public device.

Which keys are worth getting?
  • Gemini: best for truly personalized smart cards.
  • GitHub token: optional; helps avoid low unauthenticated limits while searching public repos.
  • OpenAlex: optional; scholarly papers also try to work without a key.
  • Guardian: good for book/fantasy culture news.
  • YouTube: good for learning-video discovery, but search uses quota.
  • Google Books: good for book metadata and recommendations.

Installation

Samsung setup

  1. Host the folder on a free static host such as Netlify Drop or GitHub Pages.
  2. Open the link in Chrome or Samsung Internet.
  3. Tap menu → Install app / Add page to Home screen.
  4. Open from your Home Screen, enable notifications, and import the calendar backup.
